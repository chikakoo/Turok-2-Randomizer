# Turok 2 Randomizer

A randomizer for Turok 2 Remastered for the PC, including full Archipelago multiworld integration. [Archipelago](https://github.com/ArchipelagoMW/Archipelago/releases/latest) is required to randomize the game, but isn't required to play it if you are not playing on a multiworld.
